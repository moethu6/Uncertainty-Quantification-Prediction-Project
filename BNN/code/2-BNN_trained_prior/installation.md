1. pull the Github repository of the paper "All you need is a good functional prior for Bayesian deep learning": `git clone https://github.com/tranbahien/you-need-a-good-prior.git`

2. create a python virtual environment with python 3.6 or python 3.7, recommend using the Linux environment

3. activate the virtual environment, run `pip3 install .`

4. if there are packages uncovered, using pip or conda to install individually
